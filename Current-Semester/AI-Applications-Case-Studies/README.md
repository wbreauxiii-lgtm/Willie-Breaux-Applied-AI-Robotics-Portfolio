# AI Applications Case Studies

## Overview

This folder contains current-semester AI applications case studies. The work focuses on applied AI use cases, risks, ethics, fraud detection, public-sector systems, manufacturing, finance, and facial recognition.

## Problem Statement

AI systems create value only when they are designed for real-world constraints such as fairness, accountability, privacy, misuse risk, and human oversight. These case studies examine how AI should and should not be applied in different industries.

## Tools and Technologies

- Markdown documentation
- PDF reports
- AI ethics and responsible AI frameworks
- Case-study analysis
- Public-facing portfolio documentation

## Approach / Methodology

Each case study reviews a specific domain, identifies the AI problem or opportunity, discusses possible benefits and risks, and documents responsible-use considerations.

## How to Run

This project is documentation-focused. Open the project subfolders and read each `README.md` with the matching PDF report where present.

## Results / Output

The folder includes multiple case-study README files and PDF reports. The work demonstrates applied reasoning about AI systems rather than a single executable codebase.

- How ethical, legal, privacy, and accountability issues affect AI deployment
- How to write professional documentation for applied AI topics
