# Macro Market Assistant

## Overview

This is a public-safe portfolio summary of a Natural Language Processing coursework project. The original school submission explored an AI assistant concept for organizing market and macroeconomic information.

Detailed product requirements, technical design documents, API specifications, user flows, prototype code, outputs, and strategy files are intentionally excluded from this public repository to protect private project work.

## Problem Statement

Market information can be difficult to review because news, macroeconomic signals, and financial data arrive from many sources. The project studied how an AI assistant could help organize that information for research support.

## Approach / Methodology

The coursework used a product-planning workflow: identify a user problem, define a high-level assistant concept, consider data flow and user experience, and document how the system could be evaluated.

As part of the school submission, I practiced creating AI product and technical documentation, including PRD, TRD, and TDD-style planning materials. Those detailed documents are not published here because they may contain private product strategy, architecture, prompts, data-source planning, and implementation details.
