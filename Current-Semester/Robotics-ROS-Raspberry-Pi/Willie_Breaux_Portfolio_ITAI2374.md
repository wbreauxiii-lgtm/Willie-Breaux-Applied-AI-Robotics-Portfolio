## Terms Learned
- **Raspberry Pi:** A single-board computer providing an affordable, flexible computing platform capable of running an operating system, connecting peripherals, and running applications
- **Raspberry Pi OS:** The official operating system for the Raspberry Pi, used as a desktop environment
- **Raspberry Pi Imager:** Software tool used to write Raspberry Pi OS to a microSD card for booting
- **Clustering:** Connecting multiple Raspberry Pi units together to function as a more powerful computing system
- **HAT (Hardware Attached on Top):** Additional hardware modules that integrate directly onto the Raspberry Pi's GPIO header to extend its capabilities
- **SSD (Solid-State Drive):** A storage device that can be attached to enhance the Raspberry Pi's storage speed and performance
- **Active Cooler:** A cooling device (fan) attached to the Pi to prevent overheating and ensure consistent performance

# Module MT (Midterm Project): Practical Robot Programming and Waypoint Setup — UR3e Pick-and-Place

## Activities
- Programmed the UR3e cobot to autonomously pick up blocks and transfer them onto a conveyor belt using waypoint-based motion commands
- Placed the robot in a ready position as a secure, uniform starting point
- Configured gripper open/close positions via two waypoint actions
- Set up 13 total waypoints using the Move button and Set Waypoint function in the program interface
- Used MoveL (straight-line motion) for precise pick-and-place tasks; MoveP for smoother path transitions; Circle Move available for curved motions
- Configured tool speed at 250 mm/s, TCP speed at 250 mm/s, and base acceleration at 1200 mm/s²
- Resolved a braking system issue — brakes were hindering programmed movements and had to be disabled
- Addressed a gripper release malfunction where the robot halted instead of continuing the sequence; added an additional waypoint to allow the robot to return, retrieve the block if necessary, and continue
- Willie completed all available UR training courses (except two that were unavailable)
- Willie's first day involved exploring the Cobot arm movements and manually simulating a pick-and-place from conveyor belt to cup
- Willie resolved a gripper issue with help from another team

## Results & Learnings
- The robot autonomously completed the task of moving 6 blocks onto the conveyor belt after program initiation, without further human intervention
- MoveL was particularly beneficial for accurate pick-up and placement tasks
- Properly configuring speeds and acceleration contributed to smooth, safe robot movement
- Willie learned that Cobots are powerful tools when humans understand how to guide, program, and work safely alongside them

- Willie completed UR training courses for a clearer understanding of Cobot operations and real-world use
- Time with the Cobot was cut short but still provided a wider understanding of AI and robotics in industrial manufacturing

## Terms Learned
- **Waypoint:** A programmed position/coordinate in the robot's workspace that defines a point in its motion path
- **MoveL (Linear Move):** A robot motion command that moves the tool center point in a straight line between waypoints — used for precision tasks like pick-and-place
- **MoveP (Process Move):** A robot motion command that enables smoother path transitions between waypoints, blending movements
- **Circle Move:** A robot motion command used for curved or circular motion paths
- **TCP (Tool Center Point):** The reference point at the tip of the robot's tool (e.g., center of the gripper) used to define positions and movements
- **Tool Speed:** The velocity of the robot's end effector/tool during operation (set to 250 mm/s)
- **Base Acceleration:** The rate of change of velocity for the robot's movements (set to 1200 mm/s²)
- **Pick-and-Place Operation:** An automated task where a robot picks up an object from one location and places it at another
- **Conveyor Belt:** A continuous moving surface used in manufacturing to transport objects — the target destination for robot-placed blocks
- **Gripper:** The end-of-arm tool on the robot used to grasp and release objects
- **PolyScope:** The graphical programming interface on the UR teach pendant used to create waypoint programs
- **Free-Drive Mode:** A mode allowing a human to physically guide the robot arm by hand, used for intuitive positioning and collaborative interaction
