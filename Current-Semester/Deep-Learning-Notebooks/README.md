# Deep Learning Notebooks

## Overview

This folder contains current-semester deep learning coursework notebooks and a blueprint PDF. The work covers neural-network foundations, convolutional neural networks, diffusion, transformers, and image-classification exercises.

## Problem Statement

Deep learning models require understanding how data, model architecture, training, and evaluation connect. These notebooks document practice with deep learning workflows and model-building concepts through course labs.

## Tools and Technologies

- Python
- Jupyter Notebook
- TensorFlow / Keras where used in notebooks
- PyTorch or related libraries where used in notebooks
- CNN, diffusion, and transformer concepts

## Approach / Methodology

The coursework uses notebook-based experiments. Each notebook walks through a concept or model type, with code cells, outputs, and written explanations included where present in the original files.

## How to Run

1. Install dependencies from the repository-level `requirements.txt`.
2. Open the notebooks in Jupyter or VS Code.
3. Run one notebook at a time so dependencies and data paths can be checked.
